# Questions and Answers

## University Policy & Administration

**Q: Who is the Chancellor of K.R. Mangalam University according to the student handbook?**  
A: Prof. Dinesh Singh.

**Q: What is the minimum attendance requirement to appear for the End-Term examinations at KRMU?**  
A: A minimum of 75% attendance in each course separately is required.

**Q: What is the annual hostel fee for the academic session 2023-24 at KRMU?**  
A: Rs. 1,40,000/- per year.

**Q: If a student withdraws admission before Freshmen Orientation, what percentage of the aggregate fee is refundable?**  
A: 100% refund (subject to a Rs. 2,000/- processing charge).

**Q: What primary aim does KRMU uphold besides promoting academic excellence?**  
A: Upholding moral values.

**Q: How is the SGPA (Semester Grade Point Average) calculated at KRMU?**  
A: It is the weighted average of grade points: Σ(Credit x Grade Points) / Total Credits of the Semester.

## Programming Languages & Courses

**Q: Name the four "New-Age" programming languages covered in the course ENSP415.**  
A: GO, F#, Clojure, and Kotlin.

**Q: In Kotlin programming, what is the specific purpose of the Elvis operator (?:)?**  
A: It handles nullable values by providing a default/alternative value if the expression is null.

**Q: What core components are used in the GO programming language to achieve concurrent execution?**  
A: Goroutines and channels.

**Q: What is the full form of WCF in the Advanced Topics of the .NET Framework syllabus?**  
A: Windows Communication Foundation.

## Academic & General Knowledge

### .NET Framework

**Q: What is the difference between Windows Forms and WPF?**  
A: Windows Forms uses standard Windows controls; WPF uses XAML and provides a more modern, resolution-independent UI system.

### GO Programming

**Q: What is the "gofmt" tool used for?**  
A: It automatically formats Go source code according to the language's standard style.
