# Uni-Query-AI Frontend

React + TypeScript + Vite frontend application.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Firebase
- React Router
